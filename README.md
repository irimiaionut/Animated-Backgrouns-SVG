# AnimationBackgroundSVGs

This repo contains animated SVG files.
Feel free to try and use them.


```html
<h2>Example of code</h2>

<pre>
    <div class="container">
        <div class="block two first">
            <h2>Your title</h2>
            <div class="wrap">
            //Your content
            </div>
        </div>
    </div>
</pre>
```
